About this project

This academic demo demonstrates a pipeline for detecting potentially fraudulent GSM subscriber registrations using an artificial neural network. It includes:

  • Data generation and synthetic labeling
  • Feature preprocessing and scaling
  • Model training with Keras (TensorFlow backend)
  • Simple web UI for batch predictions
References & further reading

  2. Chandola, V., Banerjee, A., & Kumar, V. Anomaly Detection: A Survey. ACM Computing Surveys.
